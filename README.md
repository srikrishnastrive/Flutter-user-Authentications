# Firebase Project

A Flutter application integrating Firebase Authentication for a robust login system.

## Overview

This project is a Flutter-based mobile application that demonstrates a complete authentication system using Firebase. It allows users to register and log in through multiple methods, providing a seamless and secure user experience.

### Features
- **User Signup**: Register with a name, email, and password.
- **Email/Password Sign-In**: Log in using registered email and password credentials.
- **Google Sign-In**: Authenticate using Google OAuth for quick access.
- **Phone Number Authentication**: Sign in with a phone number and OTP verification.
- **Firebase Backend**: Entire login mechanism powered by Firebase Authentication.

## Getting Started

This project serves as a starting point for a Flutter application with Firebase integration. Follow the steps below to set it up and explore its features.

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install) installed on your machine.
- A Firebase project set up on the [Firebase Console](https://console.firebase.google.com/).
- Android Studio, VS Code, or any preferred IDE with Flutter support.
- Basic knowledge of Dart and Flutter.
